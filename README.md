# PKU-ICS-2025Fall-Lab

北京大学计算机系统导论 2025 学年秋季学期实验集合及笔记

## 仓库结构

课程共有 8 个实验，相关实验的文件分别存放在各自的文件夹中。每个文件夹内包含：

- `README.md`：该实验的记录，包含个人的思路和实现。
- `*.pdf`：该实验的指导文档。
- `*.tar`：该实验的完整初始项目文件。
- 剩余文件为该实验的个人实现相关文件，仅供参考。
